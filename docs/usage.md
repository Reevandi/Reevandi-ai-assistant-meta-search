⚠️ This script focuses on frontend setup only.
LLM engine and meta search services are assumed

## Disclaimer
This project is developed for **educational purposes**.
It is not production-ready and is under active improvement.

## What the Script Does
- Validates required dependencies
- Deploys frontend files
- Configures PHP proxy endpoints
- Integrates with Apache web server

## Assumptions
Before running the script:
- Ollama is already installed and running
- LibreY is deployed separately
- Apache and PHP are installed

chmod +x setup-ai.sh
